# CloudflareDDNS
#### EN
>#### Requires curl grep bash crontab
>#### IPv4 Only
>
>Create DDNS using Cloudflare API
>
>It should be used with crontab.
>
> `* * * * * /home/USER/cloudflareddns.sh >/dev/null 2>&1`
